# plusコマンド
![test](https://github.com/kei487/robosys202x/actions/workflows/test.yml/badge.svg)
  標準入力から渡された数値を全て足し合わせて返却するプログラムです.

# インストール方法
```
$ git clone git@github.com:kei487/robosys202x.git
```

# コードの実行方法
```
$ chmod +x plus
$ seq 10 | ./plus 
55
```
## 必要なソフトウェア
* Python
   *  テスト済み: 3.7~3.10

## テスト環境
* Ubuntu
